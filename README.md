# dcn_game

Online board game eco4 for DCN

## Requirements

- Flutter ^3.3.9

## Installation

- Clone the repo
- Run `flutter pub get` to install dependencies

## Run the client server
- Run `flutter run` to start the app

## Run the API server
- Run with dart the file ./lib/server/server.dart
